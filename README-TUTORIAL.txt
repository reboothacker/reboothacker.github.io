Настройка панели Telegram BOT
Все журналы приходят внутри телеграмм-бота, и вы можете настроить его следующим образом.
1. Сообщение @botfather в телеграмме
( t.me/botfather )
2. Создайте такого бота
2.1 выберите или введите /newbot
2.2 Введите имя, например: Randomname
2.3 Введите имя пользователя, например: yourname_bot
3. Вы получите токен, который выглядит следующим образом:
11111111111111:euuxbeiwososbx1626gdhdvsbdjdhdhdhhs
4. Скопируйте этот токен
5. Найдите в файлах страницы файл /config/config.php и откройте его.
(Вы можете открыть файл с помощью notepad++ или любой другой текстовый/кодовый редактор)
 6. Ищем строчку, где можно увидеть /* токен бота */
 (строка 2)
 7. Вставьте токен между ""
 8. Отправьте сообщение (/start) на адрес t.me/userinfobot или t.me/useridinfobot.
 9. Вы получите свой идентификатор пользователя телеграммы
 10. Скопируйте этот код
 11. Снова откройте файл config.php
 12. Найдите строку, которая говорит /* chatid */
 (строка 3)
 13. Введите свой идентификатор пользователя между ""
 14. Ваш код в config.php должен выглядеть так:
 15. Отправьте сообщение своему боту или нажмите /start, и ваш бот готов к работе.

 2. Кнопка бана

 В телеграмм боте есть кнопка бана и разбана она блокирует айпи лога. Разбан
 кнопка не разбанивает выбранный ip, а очищает весь банлист из-за ошибки, допущенной пользователем
 кодер, так что лучше всего оставить забаненный ip после блокировки или удалить его вручную из ips.txt файл
